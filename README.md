# SmartFlow

Крипто-аналітика: Telegram бот + Mini App. SMC-сетапи, скоринг за
конфлюенсом, новинний сентимент (Gemini), алерти, графіки з розміткою зон,
автотрейдинг (non-custodial) на тарифі Auto.

- Специфікація: docs/SPEC.md
- Інструкція для Claude Code: CLAUDE.md
- Плейбук задач: docs/PROMPTS.md
- Журнал рішень: docs/DECISIONS.md

1. git init && git add . && git commit -m "project skeleton"
2. Установи Claude Code (PowerShell): irm https://claude.ai/install.ps1 | iex
